Demonstration of Different CSS Selectors

Using Class, ID, Descendant, Group and Universal Selectors

1. Class Selector Example

This paragraph is styled using a class selector with the class name "highlight".

Multiple elements can share the same class and will have the same style.

2. ID Selector Example

This paragraph uses an ID selector with the ID "special". IDs are unique and used only for one element on a page.

3. Descendant Selector Example

This paragraph is inside a div, so it is styled using the descendant selector (div p).

Descendant selectors help style elements based on their hierarchy.

4. Group Selector Example

kaushal kumar gupta (H1)

kashinath kumar(H2)

kunal mandle (H3)

All these headings share common styles using the group selector (h1, h2, h3).

5. Universal Selector Example

The universal selector (*) applies basic styling (font, margin, padding) to every element on this web page.
