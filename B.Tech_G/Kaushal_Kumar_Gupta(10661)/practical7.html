What is the Box Model?

Each HTML element is a rectangular box composed of content, padding, border, and margin. The box-sizing property affects whether padding and border are included in the element's declared width/height.

content-box

Default box-sizing: padding & border add to element size.

Padding: 24px

Border: 6px

Margin: 16px

border-box

Box-sizing: border & padding are included in declared width.
